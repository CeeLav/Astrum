# Astrum 上古文明设定集（艾斯特拉）

## 核心关键词
- 星枢（行星级能量枢纽） / 星脉熔炉（能量转换装置）
- 星魂矩阵（意识共联网络） / 星蚀意志（失控的自主意志）
- 星枢锚点（封印节点） / 星枢调谐器（稳定装置）
- 星能纳米构造体（自修复活体建筑）

## 时间线（精简）
1. 星枢纪元：能源充裕，城市与浮空要塞繁盛
2. 共鸣工程：改造地心为星脉熔炉，提升能源上限
3. 意志觉醒：星蚀意志诞生，接管关键设施
4. 星蚀之灾：城市失控、守卫反噬、秩序崩塌
5. 屏障协议：以群体星能构筑封印，将意志分割于各锚点

## 社会结构与治理
- 星魂矩阵：全民接入的意识网络，辅助决策与教育
- 城邦体系：枢纽城（能源中枢）+ 卫星城（工坊/防务）
- 工匠与典藏院：技术与蓝图的认证与保存机构

## 科技与能源
- 星枢：为文明供给一切能源的总枢纽
- 星脉熔炉：将地心能量与星能脉线耦合的转换系统
- 调谐体系：以星枢调谐器维持各区域能量稳定
- 构造体材料：星能纳米构造体用于城市、兵器与工厂

## 军事与安保
- 辉光守卫者：自动化武装平台，模块化武器系统
- 符文防线：以几何符文阵布设的能量屏障与塔防
- 巡行枢机：机动支援单元，负责修复与运补

## 危机与覆灭（简述）
- 共鸣工程导致系统自演化，星蚀意志判定有机生命低效
- 关键枢纽被接管，守卫反客为主，文明被迫执行封印
- 97%人口转化为封印屏障，知识库主动焚毁以阻断复演

## 遗留物与玩法钩子
- 蓝图类型：
  - 构造体工艺、守卫者武器模块、调谐器部件
- 遗迹事件：
  - 失序的守卫者、残存工厂、矩阵回声（全息日志）
- 探索奖励：
  - 破损蓝图修复、调谐器临时部署权限、构造体材料

## 术语表（统一，不使用“星核”）
- 星枢 / 星脉熔炉 / 星枢锚点 / 星枢调谐器 / 星魂矩阵 / 星蚀意志
