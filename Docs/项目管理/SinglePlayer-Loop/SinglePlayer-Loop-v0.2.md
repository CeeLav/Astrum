# 单机闭环 v0.2 计划（简版）

## 目标

- 在探索中加入“塔防事件”骨架：核心HP/波次/插槽激活。

## 范围

- 在范围：塔防事件触发、事件HUD（波次/核心HP）、插槽激活/升级（消耗碎片）。
- 不在范围：自由建造、全局昼夜、复杂路径寻路。

## 里程碑（+2–3天）

- 事件可玩，成功/失败均正确结算。

## 任务清单

- 事件触发与结束条件（成功/失败）。
- 核心HP与波次控制、事件HUD。
- 插槽激活/升级与消耗（简化规则）。
- 与 v0.1 流程对接：事件内奖励 → 结算 → Hub。

## 验收标准

- 成功/失败两条路径均能正常结算。
- 事件奖励对下一轮有感知提升。

## 风险与回退

- 事件难度与时长：先以固定参数上线，后续调优。
- 与主循环耦合：事件逻辑独立，接口化对接结算。
